README for Manifold

Manifold is a FreeFrame plugin (FFGL, version 1.5) that can be used within Resolume 4.0. (It's possible that it may work in Resolume 3.3 as well.) It is capable of listening for TUIO over OSC and generating graphics and music from it. It was originally designed for use with the Space Palette, but can also be used with any TUIO-generating application, such as TuioPad.

Getting Started

After installing Manifold, you need to configure Resolume in the following way:
  1. Enable OSC Input on port 7000 (the default)

  2. Add the following directory to the list of FreeFrame plugin directories:

    c:\Users\Public\Documents\Nosuch Media\FFGLPlugins 

Browser-based User Interface

After you start Resolume, Manifold should appear as a plugin in the Effects section. When you add Manifold to a composition and run it, you can interact with it by pointing a browser at http://127.0.0.1:4445. You can also access this interface remotely by using the IP address or hostname of the computer, rather than 127.0.0.1 (localhost). The browser interface only works if Resolume is running and you are running a clip that has Manifold as an effect plugin.
