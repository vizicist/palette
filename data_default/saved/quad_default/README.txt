This directory holds the "Default" pro2 quad theme.

Each .json file here is a link to a preset in the sibling "quad" directory,
which is the master set of quad presets. Themes are selected with the pro2
Theme Selector; a theme is a curated subset of the master presets.

On first launch the Default theme is seeded with links to every quad preset
(recorded by a .seeded marker so it is only seeded once). This README keeps the
directory present in the installer.
